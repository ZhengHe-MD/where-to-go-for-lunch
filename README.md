# 中午去哪吃

[English](./README-en.md)

在鼎好工作时，中午总是不知道该去哪家店吃饭。于是就开发了这个小应用，每天中午吃饭前转一下。

如今那段日子已经过去很长时间，最近清理代码时忽然发现这个项目，觉得还挺有意思，就放到 github 上保存着，免得日后找不到。

如果你有兴趣，戳 [这里](https://bit.ly/2Wll7Qi) 试试

## 功能

* 支持自定义 3 - 12 个选择类别
* 支持模板保存和加载
* 支持通过 url 分享给别人
